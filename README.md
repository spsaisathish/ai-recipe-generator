# AI Recipe Generator

Enterprise GenAI Recipe Generator built using:

- Next.js
- NestJS
- TypeScript
- Claude AI
- Prompt Engineering
- PostgreSQL
- Prisma

## Architecture

Frontend (Next.js)
        ↓
Backend (NestJS)
        ↓
AI Service
        ↓
Prompt Builder
        ↓
Claude API
